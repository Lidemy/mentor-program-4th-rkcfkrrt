## 請找出三個課程沒教的 HTTP status code 並簡單介紹
 
1. 304 Not Modified：表示資源內容和上次一樣，可以從快取取得副本即可。
2. 403 Forbidden：表示用戶端沒有權限存取、被禁止使用，因此伺服器拒絕回應。IIS 有更詳細定義一些不同錯誤原因的 403。
3. 504 Gateway Timeout：表示閘道逾時，伺服器沒有回應。
